# Advanced-todo-card#

This project is an upgraded version of a simple Todo Card.  
It adds more interactivity, editable content, and better state handling to make it feel more like a real application component.

## What Changed from Stage 0

- Added edit mode for updating task details
- Added status control (Pending, In Progress, Done)
- Improved priority display with visual indicator
- Added expand/collapse for long descriptions
- Improved time remaining logic with overdue handling
- Introduced basic state management for the card


## Features

- View task details (title, description, priority, due date)
- Edit task information using a form
- Save or cancel changes in edit mode
- Toggle task status using checkbox or dropdown
- Expand and collapse task description
- Dynamic time remaining updates
- Overdue indicator when deadline passes
- Visual priority indicator (color-based)


## How to Run

1. Clone the repository
2. Open the folder
3. Open `index.html` in your browser

No installation required.


## Accessibility Notes

- Uses semantic HTML elements like article, time, and button
- Includes proper labels for form inputs
- All interactive elements are keyboard accessible


## Known Limitations

- Data is not saved after refresh
- No backend or database connection
- Edit/Delete actions are only local to the page


## Design Decisions

- Built using vanilla HTML, CSS, and JavaScript for simplicity
- Focused on clean UI and easy readability
- Kept logic lightweight without frameworks
- Used basic state management inside JavaScript for control
